Общие настройки
Шаблон имени ГОСТа  

Доступны переменные:

%номер - номер ГОСТа, например "31610.5"

%номер_год - номер ГОСТа, включая год, например "31610.5-2012"

%имя - наименование ГОСТа, например "Кефир. Технические условия"

Руководство пользователя
1. Нажмите на иконку «1» напротив наименования ГОСТа для начала загрузки страниц и вывода на печать в формате PDF
2. В случае появления предупреждения о блокировке всплывающих окон, нажмите на кнопку для настройки блокировки и выберите опцию «Всегда показывать всплывающие окна с сайта protect.gost.ru». Нажмите кнопку «Готово» и перезагрузите страницу кнопкой для применения изменений. Перейдите к п.1. для повторного сохранения ГОСТа
3. В диалоговом окне печати нажмите кнопку «Изменить» (3) выберите принтер «Сохранить как PDF» если он не был выбран ранее.
4. Настройте параметры вывода изображения в разделе «Настройки» (4), вы можете выключить вывод информации в колонтитулы страницы.
5. Нажмите кнопку «Сохранить» для сохранения ГОСТа в PDF.
6. Выберите папку для сохранения ГОСТа и укажите название файла. Оно автоматически формируется расширением на основе наименования ГОСТа.
7. Нажмите кнопку «Сохранить» для сохранения файла на диск. Теперь вы можете открыть его в любой программе просмотра PDF-файлов.
